## [1.0.0] - 2024-11-13

### Added
- 初始版本发布
- 支持在 JetBrains IDE 中打开微信读书
- 使用 JCEF 内嵌浏览器显示微信读书网页版
- 支持完整的微信读书功能
- 添加中文和英文国际化支持
- 添加应用程序激活监听器

### Features
- 工具窗口集成，可在 IDE 右侧打开
- 自适应窗口大小
- 支持所有 JetBrains IDE（Android Studio、IntelliJ IDEA、PyCharm 等）
- 支持 IDE Build 223 到 251（包括最新的 Android Studio AI-251）

### Fixed
- 更新 pluginUntilBuild 到 251.* 以支持最新版本的 Android Studio
